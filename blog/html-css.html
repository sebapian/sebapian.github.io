Display Inline vs Inline Block

2/07/2022

Introduction

The display CSS property sets how an element looks. It has a significant impact on layouts of the children elements. There are different types of display properties, a couple include: inline, inline-block, block, grid, table. Each of these display properties have their respective uses and layouts. Today, let's dive into what the differences are between Inline vs Inline-block.

display: inline

With display: inline, it integrates the child element in the same line as the content in the parent element. However, as you can see, the top and bottom margins & paddings are not respected. The width and height of the child element are also not respected.

display: inline-block

display: inline-block enables you to create side by side boxes that collapse and wrap properly. With display:inline, the height and padding are respected, but the two child element can still sit side by side.
